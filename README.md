# Restaurant Intelligence System

## Overview

Restaurant Intelligence System is an end-to-end Data Science and Machine Learning project built using Python, Streamlit, Scikit-learn, and Data Visualization libraries.

The project analyzes restaurant data to generate business insights, predict restaurant ratings, provide intelligent restaurant recommendations, perform cuisine exploration, and visualize restaurant locations interactively.

The system combines analytics, machine learning, recommendation systems, and geospatial analysis into a single interactive dashboard.



# Objectives

- Perform restaurant data exploration and preprocessing
- Analyze restaurant trends and customer preferences
- Predict restaurant ratings using Machine Learning
- Build a real-time restaurant recommendation system
- Perform cuisine-based restaurant analysis
- Visualize restaurant locations using interactive maps
- Develop an end-to-end deployable analytics dashboard

The project uses the Zomato Restaurant Dataset containing:
- Restaurant names
- Cities
- Cuisines
- Aggregate ratings
- Votes
- Price ranges
- Latitude and longitude
- Online delivery information
- Table booking information
